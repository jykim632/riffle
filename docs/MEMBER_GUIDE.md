# Riffle 스터디 회원가입 가이드

## 접속 주소

**https://web.riffles.cloud**

---

## 회원가입 방법

두 가지 방법 중 하나를 선택하면 됩니다.

### 방법 1: 이메일로 가입

1. https://web.riffles.cloud/signup 접속
2. 아래 정보 입력:
   - **초대 코드**: 단톡방에서 받은 8자리 코드 (대문자+숫자)
   - **닉네임**: 1~20자 (한글, 영문, 숫자, _ 사용 가능)
   - **이메일**: 본인 이메일
   - **비밀번호**: 8자 이상
3. **회원가입** 버튼 클릭
4. 완료되면 대시보드로 자동 이동

### 방법 2: Google 계정으로 가입

1. https://web.riffles.cloud/signup 접속
2. 하단의 **Google로 가입** 버튼 클릭
3. **초대 코드** 입력 (8자리)
4. **Google로 계속하기** 클릭
5. Google 계정 선택 후 로그인
6. 완료되면 대시보드로 자동 이동

> 초대 코드는 1회용이므로, 본인에게 배정된 코드를 사용해주세요.

---

## 로그인

가입 후 재방문 시:

1. https://web.riffles.cloud/login 접속
2. 가입 시 사용한 이메일/비밀번호 입력, 또는 **Google로 로그인** 클릭

---

## 주요 기능

### 대시보드

로그인하면 바로 보이는 화면입니다.

- 이번 주차 정보 (기간, 제목)
- 내 제출 여부
- 전체 멤버 제출 현황

### 요약본 제출

1. 상단 메뉴에서 **새 요약본** 클릭
2. 주차 선택 (현재 주차가 기본 선택)
3. 마크다운 에디터로 요약본 작성
4. **제출** 클릭

### 요약본 조회

- 다른 멤버들의 요약본을 목록에서 확인 가능
- 본인 요약본은 수정 가능

---

## 문제가 생겼을 때

| 증상 | 해결 |
|------|------|
| 초대 코드가 안 먹혀요 | 대소문자 확인 (대문자+숫자만), 이미 사용된 코드인지 확인 |
| 비밀번호 오류 | 8자 이상인지 확인 |
| 닉네임 오류 | 특수문자(!, @, # 등) 제거, _ 만 허용 |
| Google 로그인 안 돼요 | 팝업 차단 해제 후 재시도 |
| 그 외 | 단톡방에서 문의 |
