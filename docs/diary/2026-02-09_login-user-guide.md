## 작업 내용

### 사용자 가이드 모달 (`UserGuideDialog`)
- `src/components/auth/user-guide-dialog.tsx` 신규 생성
- shadcn/ui Dialog 기반, 3개 섹션 구성:
  - **서비스 소개**: Riffle이 뭔지 한줄 설명 (blue-violet gradient 배경)
  - **이용 방법**: 초대코드 → 회원가입 → 요약 제출 (번호 뱃지 스타일)
  - **FAQ**: 초대코드 수령, 비밀번호 분실, Google 로그인 관련 3문항

### 로그인 페이지 수정
- `src/app/(auth)/login/page.tsx`에 `UserGuideDialog` 배치
- CardFooter 내 회원가입 링크 바로 아래 "처음이신가요? 사용 가이드 보기" 텍스트 링크

## 왜 했는지

- 처음 방문하는 사용자 입장에서 Riffle이 뭔지, 어떻게 가입하는지 알 수 없음
- 폐쇄형 서비스라 초대코드 개념 자체가 낯설 수 있음
- 별도 랜딩 페이지 없이 로그인 페이지에서 바로 안내하는 게 가장 경제적
