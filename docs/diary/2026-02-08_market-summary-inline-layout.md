# 시장 요약 위젯 한 줄 인라인 레이아웃

**날짜**: 2026-02-08
**커밋**: a7176de

## 작업한 내용

`market-summary-widget.tsx`의 레이아웃을 그리드(2x2 → 4열)에서 flex 인라인 한 줄로 변경.

- `grid grid-cols-2 sm:grid-cols-4` → `flex items-center overflow-x-auto`
- 각 지표 항목: 세로 스택(라벨/값/변동) → 가로 인라인(라벨 값 변동)
- 지표 간 `|` 구분선 추가
- 모바일: `overflow-x-auto` + `shrink-0`으로 가로 스크롤 처리

## 왜 했는지

대시보드 시장 요약 카드가 2x2 그리드로 세로 공간을 많이 차지하고 있었음. 지표 4개(환율, KOSPI, 기준금리, CPI)는 한 줄에 충분히 들어가는 양이라 컴팩트하게 변경. 이미 `indicators-ticker.tsx`가 동일한 인라인 패턴을 쓰고 있어서 일관성도 맞춤.

## 결정된 내용

- 티커 바와 동일한 flex 인라인 패턴 채택
- 모바일에서는 가로 스크롤 (wrap 아닌 scroll) — 지표가 잘려 보이는 것보다 스크롤이 낫다고 판단
- `tabular-nums` 유지해서 숫자 정렬 깔끔하게

## 난이도/발견

- 난이도: 낮음 (단순 레이아웃 변경)
- 기존 티커 바 패턴이 있어서 거의 그대로 적용
- `p` 태그를 `span`으로 바꿔야 flex 인라인이 자연스러움
