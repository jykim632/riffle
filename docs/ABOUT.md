# Riffle - 경제 라디오 스터디

> 매주 경제 라디오 요약본을 작성하고 공유하는 스터디 플랫폼

---

## Riffle이 뭔가요?

매주 경제 라디오를 듣고, 핵심 내용을 요약해서 올리는 스터디를 위한 전용 웹앱입니다.
듣기만 하면 흘러가는 내용을, 직접 정리하고 서로 비교하면서 더 깊이 이해하는 게 목표입니다.

---

## 주요 기능

### 대시보드

로그인하면 바로 보이는 메인 화면입니다.

- **이번 주 현황**: 현재 주차 정보, 기간, 내 제출 여부
- **멤버 제출 현황**: 누가 제출했는지 한눈에 확인
- **이번 주 요약본**: 다른 멤버들이 올린 요약본 미리보기

### 요약본 작성

- 마크다운 에디터로 자유롭게 작성
- 실시간 미리보기 지원
- 시즌 내 주차를 선택해서 제출
- 제출 후에도 수정 가능 (버전 히스토리 보관)

### 게시판

- 전체 요약본을 주차별로 필터링해서 조회
- 다른 멤버의 요약본을 읽고 비교
- 내 제출 내역만 모아보는 페이지 제공

### 시즌 시스템

- 약 3개월 단위로 시즌 운영
- 시즌마다 멤버 구성 가능
- 주차는 매주 월~일 자동 전환

### 초대제 운영

- 초대 코드가 있어야 가입 가능
- 스터디 멤버만 참여하는 폐쇄형 플랫폼

---

## 지원 환경

- PC, 태블릿, 모바일 모두 지원 (반응형)
- 다크 모드 지원
- 이메일 또는 Google 계정으로 로그인

---

## 앞으로 추가될 기능

### 곧 추가 예정

| 기능 | 설명 |
|------|------|
| 비밀번호 재설정 | 비밀번호를 잊었을 때 이메일로 재설정 |
| UI 개선 | 전반적인 사용성 다듬기 |

### 계획 중

| 기능 | 설명 |
|------|------|
| 댓글/피드백 | 다른 멤버의 요약본에 한줄 코멘트 남기기 |
| 제출 리마인더 | 미제출 시 알림 (이메일 또는 웹 푸시) |
| 참여 통계 | 시즌별 제출률, 연속 제출 기록 등 |
| 요약본 검색 | 키워드로 과거 요약본 검색 |
| 프로필 편집 | 닉네임 변경, 아바타 설정 |
| 시즌 아카이브 | 지난 시즌 요약본 모아보기 |
| 베스트 요약본 | 주차별 좋아요/추천 기능 |
